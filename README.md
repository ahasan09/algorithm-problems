# Algorithm Problems

A collection of classic algorithm implementations in JavaScript, each solving a specific problem category.

## Problems

| Folder | Description |
|--------|-------------|
| `anagrams` | Find all anagrams of a word from a word list |
| `frequency-counter` | Count character/element frequencies |
| `hanoi-tower` | Tower of Hanoi recursive solution |
| `number-to-word` | Convert numbers to their English word representation |
| `rock-paper-scissor` | Rock-Paper-Scissors game logic |
| `rolling-dice` | Dice roll simulation and statistics |

## Prerequisites

- A modern browser, or
- [Node.js](https://nodejs.org/) v14+

## Running in the Browser

Most problems have an `index.html` file. Simply open it in a browser:

```bash
# macOS
open anagrams/index.html

# Linux
xdg-open anagrams/index.html

# Windows
start anagrams/index.html
```

## Running with Node.js

```bash
node anagrams/index.js
node hanoi-tower/index.js
node number-to-word/index.js
```
