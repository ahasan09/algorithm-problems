# Algorithm Problem

Collection of 6 vanilla JavaScript mini-projects and algorithm exercises, each runnable directly in a browser.

## Tech Stack
- Vanilla JavaScript (ES6+)
- HTML / CSS
- No build tools or package manager

## Project Structure
```
algorithm-problem/
├── anagrams/
├── frequency-counter/
├── hanoi-tower/
├── number-to-word/
├── rock-paper-scissor/
└── rolling-dice/
```
Each subfolder contains its own `index.html` and associated JS/CSS files.

## Development
No installation required. Open any subfolder's `index.html` directly in a browser:
```bash
open algorithm-problem/<subfolder>/index.html
```

## Key Notes
- No dependencies or build step — each exercise is fully self-contained.
- Projects cover: anagram detection, frequency counters, Tower of Hanoi, number-to-word conversion, rock-paper-scissors game, and a dice roller.
